# 🚀 Smart Productivity Dashboard

A modern and professional Productivity & Attendance Management Mini Project built using HTML, CSS, JavaScript and Chart.js.

---

# 📌 Project Overview

This project helps students manage:

- Daily Tasks
- Attendance Records
- Productivity Tracking
- Task Completion Status

The system includes a secure Register/Login feature and a modern dashboard UI.

---

# ✨ Features

## 🔐 Authentication System
- User Registration
- User Login
- Local Storage Authentication

## 📋 Task Management
- Add Tasks
- Edit Tasks
- Delete Tasks
- Mark Tasks Completed

## 📅 Attendance Tracking
- Daily Attendance Record
- Task History Tracking

## 📊 Dashboard Analytics
- Total Tasks
- Completed Tasks
- Pending Tasks
- Pie Chart Visualization

## 🌙 UI Features
- Dark Mode
- Responsive Design
- Glassmorphism UI
- Mobile Friendly Layout

# 🛠️ Technologies Used
